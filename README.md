# 📺 Global YouTube Statistics Analysis Dashboard

An interactive **Power BI dashboard** designed to analyze global YouTube channel performance across countries, content categories, subscriber counts, uploads, video views, and estimated earnings. The dashboard helps content creators, marketers, and analysts uncover trends in the creator economy and identify the most successful channels and content categories worldwide.

---

# 📖 Description / Purpose

The **Global YouTube Statistics Analysis Dashboard** is an interactive Power BI report that transforms global YouTube statistics into meaningful business insights. It enables users to explore creator performance, compare countries and categories, evaluate audience engagement, and understand revenue trends through dynamic visualizations and filters.

---

# 🛠 Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Used to build interactive dashboards and visual reports.
* 📂 **Power Query** – Cleaned and transformed raw YouTube statistics data.
* 🧠 **DAX (Data Analysis Expressions)** – Created KPIs, calculated measures, and dynamic metrics.
* 📝 **Data Modeling** – Built relationships between tables to enable efficient filtering and analysis.
* 🎛️ **Interactive Slicers** – Country filter enables dynamic exploration of all dashboard visuals.
* 📁 **File Format** – `.pbix` for development and `.png` for dashboard preview.

---

# 📂 Data Source

**Source:** Kaggle – Global YouTube Statistics Dataset

The dataset contains information on leading YouTube channels worldwide, including:

* Country
* Channel Name
* Category
* Subscribers
* Video Views
* Number of Uploads
* Estimated Monthly Earnings
* Estimated Yearly Earnings
* Average Views Per Upload

The dataset was cleaned, transformed, and modeled using Power Query before visualization in Power BI.

# ✨ Features / Highlights

## 📌 Business Problem

With thousands of successful YouTube creators across different countries and categories, it is challenging to understand what drives channel growth and revenue.

Important business questions include:

* Which countries have the highest-performing YouTube channels?
* Which content categories receive the most views?
* Which creators upload the most content?
* Who earns the highest estimated revenue?
* Is there a relationship between uploads and video views?
* Which content categories attract the largest subscriber base?

Without an interactive dashboard, answering these questions requires extensive manual analysis.

---

## 🎯 Goal of the Dashboard

The dashboard was designed to:

* Analyze YouTube channel performance globally.
* Compare creator statistics across countries.
* Identify the most popular content categories.
* Discover top-performing creators based on uploads, views, and earnings.
* Support content strategy and marketing decisions using data-driven insights.

# 📈 Business Impact & Insights

### 📍 Content Strategy

Music and Entertainment consistently generate the highest viewership, helping creators identify high-demand content niches.

### 📍 Creator Benchmarking

The dashboard enables comparison of leading creators based on uploads, audience reach, and estimated earnings, providing valuable competitive insights.

### 📍 Revenue Analysis

Estimated earnings reveal which creators and categories generate the greatest monetization potential, supporting sponsorship and investment decisions.

### 📍 Audience Engagement

The relationship between uploads and views helps determine whether publishing frequency translates into higher audience engagement.

### 📍 Geographic Analysis

Country-level filtering enables businesses, agencies, and creators to understand regional trends and tailor content strategies for specific markets.
